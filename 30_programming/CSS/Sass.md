# Sass（サス）
- CSSをより便利かつ効率的に書くことができる言語
  - コード量を減らせる
  - コードを再利用できる
- SASS記法とSCSS記法の2種類の書き方があり
# Scss記法
ファイル拡張子も「.css」ではなく「.scss」
# 入れ子構造
- 1つのセレクタの中に別のセレクタを入れ子にして指定することができます。これにより、同じセレクタを何度も書かなくて良いことになる
- より大規模なサイトになるほど効果が大きくなる
  - 例：headerというクラス名を変えたい場合、CSSでは3箇所変える必要がありますが、Sassでは1箇所の変更で済ませることができる
## &記号：hover系
入れ子構造を使うことによって、「セレクタ:hover」を「&:hover」というように書き換えることができます。この「&」はhoverだけでなく、activeなどにも使うことができる
## &記号：特定要素の指定
```sass
&.クラス名
```
# 変数
変数の定義
- 変数は利用する箇所より前で定義しないと使えません
- スタイルシート（CSS）は上から下に読み込まれていく
- 変数は利用できる範囲も指定できます。この利用できる範囲のことを「スコープ」と呼ぶ
```sass
$変数名: 値;
```
変数の使い方
- 変数名（$progate-color）を記述するだけ
```sass
color:$progate-color;
```
# mixin
いくつかのコードを1つにまとめて、複数箇所で簡単に呼び出すことができる機能
- mixinを定義する
```sass
@mixin mixin名 {
  内容
}
```
定義したmixinを使う方法
- 使用したいSassファイルの中で「@include mixin名」とすることで使う
```sass
.クラス名{
  @include mixin名;
}
```
## 引数を使ってmixinにデータを渡す
- mixinを定義する
```sass
@mixin mixin名($引数名) {
  内容:引数名;
}
```
定義したmixinを使う方法
```sass
.クラス名{
  @include mixin名(引数で処理してほしい内容。色指定とか);
}
```
# 関数
関数を利用すると「特定の色から50%明るい色を使う」といったことが簡単にできる
- 様々な関数が用意されている
- よく使われる関数は、色を操作する関数
```sass
  .blue2 {
    // 色を$blue、値を10%としてdarken関数を用いてください
    background-color:darken($blue, 10%);
    
  }
  
  .pink2 {
    // 色を$pink、値を10%としてlighten関数を用いてください
    background-color:lighten($pink, 10%);
    
  }
```
