# 基本
# HTML文書の宣言をする
```HTML
<!DOCTYPE html>
<html>
  <head>
    <title>My Page</title>
  </head>
  <body>
  </body>
</html>
```
- <!DOCTYPE>宣言がないと、ブラウザはHTMLの解釈を誤ることがあります。例えば、CSSが正しく適用されず、レイアウトが崩れたり、フォントサイズが異なったりします。また、JavaScriptが期待通りに動作しない場合もあります。これにより、ページが意図した通りに表示されず、ユーザー体験が損なわれることがあります。
- HTML5は、<!DOCTYPE html>の1種類のみ
- <html lang="ja">のようにlangで言語を指定可能
# <head>タグ
```HTML
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width,initial-scale=1" />
  <title>My Page</title>
  <link rel="stylesheet" href="css/style.css" />
</head>
```
- <head>タグは、ページの情報、タイトル、CSSなどの外部ファイルを読み込むリンクなど、ページに関する情報を記述する場所です。
- <head>タグ内の情報はページを見るユーザーに直接表示されることはありませんが、検索エンジンやブラウザに伝わるため、ページを公開する上で重要な役割を持っています。
- charset：ページで使用される文字の種類（文字コード）を指定します。
- viewport：ページをレスポンシブデザイン（スマートフォンやタブレットで見やすくする）に対応させる指定です。
- <title>：ページタイトルの指定です。
  - 定義したタイトルは検索結果に表示されるだけでなく、ブックマークやSNSでURLを共有した際のタイトルとしても表示されます。
  - ページ内容が伝わるシンプルなタイトルにすることは、SEO対策にも有効です。
  - 一般的には、検索エンジンやSNSでの表示において、タイトルは50〜60文字程度が推奨されています。これを超えると、切り捨てられることがあります。
- <link>：スタイルシート（CSSファイル）などを読み込みます。href属性には、適用したいファイルパスを指定します。
  - 基本的に<head>タグ内に記述します。
  - rel属性：relation（関係）の略で、タグを設置するページと関連づけるファイルとの関係性を示します。
  - href属性：関連づけるファイルのURLを記述します。
  - ブラウザのタブやブックマークに表示するアイコン画像（ファビコン）を<link>タグで関連づけています。
    - rel属性：icon
    - href属性：アイコン画像のURL
# 装飾
## <span>タグ
<span>タグで囲まれたテキストに、特定のスタイルを適用
- 文字に下線を引く
```HTML
<span>対象</span>
```
```HTML
<style>
  span{
 text-decoration: underline;
}
</style>
```
