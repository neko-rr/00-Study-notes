# 基本
# HTML文書の宣言をする
```HTML
<!DOCTYPE html>
<html>
  <head>
    <title>My Page</title>
  </head>
  <body>
  </body>
</html>
```
- <!DOCTYPE>宣言がないと、ブラウザはHTMLの解釈を誤ることがあります。例えば、CSSが正しく適用されず、レイアウトが崩れたり、フォントサイズが異なったりします。また、JavaScriptが期待通りに動作しない場合もあります。これにより、ページが意図した通りに表示されず、ユーザー体験が損なわれることがあります。
- HTML5は、<!DOCTYPE html>の1種類のみ
- <html lang="ja">のようにlangで言語を指定可能
# <head>タグ
```HTML
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width,initial-scale=1" />
  <title>My Page</title>
  <link rel="stylesheet" href="css/style.css" />
</head>
```
- <head>タグは、ページの情報、タイトル、CSSなどの外部ファイルを読み込むリンクなど、ページに関する情報を記述する場所です。
- <head>タグ内の情報はページを見るユーザーに直接表示されることはありませんが、検索エンジンやブラウザに伝わるため、ページを公開する上で重要な役割を持っています。
- charset：ページで使用される文字の種類（文字コード）を指定します。
- viewport：ページをレスポンシブデザイン（スマートフォンやタブレットで見やすくする）に対応させる指定です。
- <title>：ページタイトルの指定です。
  - 定義したタイトルは検索結果に表示されるだけでなく、ブックマークやSNSでURLを共有した際のタイトルとしても表示されます。
  - ページ内容が伝わるシンプルなタイトルにすることは、SEO対策にも有効です。
  - 一般的には、検索エンジンやSNSでの表示において、タイトルは50〜60文字程度が推奨されています。これを超えると、切り捨てられることがあります。
- <link>：スタイルシート（CSSファイル）などを読み込みます。href属性には、適用したいファイルパスを指定します。
  - 基本的に<head>タグ内に記述します。
  - rel属性：relation（関係）の略で、タグを設置するページと関連づけるファイルとの関係性を示します。
  - href属性：関連づけるファイルのURLを記述します。
  - ブラウザのタブやブックマークに表示するアイコン画像（ファビコン）を<link>タグで関連づけています。
    - rel属性：icon
    - href属性：アイコン画像のURL

# セクショニングコンテンツ
Webページ内のHTML文書を、ヘッダーやフッター、ナビゲーション、記事などといったようにパーツ毎に分ける役割を持ちます。
検索エンジンに構造が正確に伝わることで、Webページの検索順位が向上する可能性があるため、SEO対策にも効果的です。
## header
主にタイトル・ロゴ・ナビゲーションメニュー(リンク集)・検索フォーム
```HTML
<header>
  <h1>My Page</h1>
</header>
```
## nav
ナビゲーションメニューを表すために使用します。
主に 、ページ内の目次としての役割を持ちます。
```HTML
<nav>
  <ul>
    <li><a href="#">ホーム</a></li>
    <li><a href="#">サービス</a></li>
    <li><a href="#">お問い合わせ</a></li>
  </ul>
</nav>
```
## main
ページ内で中心となる内容となるコンテンツを表します。
1ページに1回だけ使用でき、ヘッダーやフッター、ナビゲーション、サイドバーなどの補足コンテンツは含みません。
```HTML
<main>
  <h1>HTMLの講座</h1>
  <p>HTMLについて、初心者にわかりやすくお教えします。</p>
</main>
```
## article
独立していて、その内容だけで完結するコンテンツであることを表します。
ブログ投稿やニュース記事などが該当します。
```HTML
<article>
  <h2>ブログタイトル</h2>
  <p>投稿日：2024年1月1日</p>
  <p>ブログ本文です。</p>
</article>
```
## section
ページ内の主要なコンテンツをグループ化するために使用されます。
- <main>や<article>タグ内の各コンテンツを囲むことが一般的です。
```HTML
<main>
  <section>
    <h2>私について</h2>
    <p>自己紹介セクションです。</p>
  </section>
  <section>
    <h2>サービス</h2>
    <p>サービス紹介セクションです。</p>
  </section>
</main>
```
## aside
「余談」や「補足情報」の意味を持ち、ページのメインコンテンツから独立していて、本文に影響しないセクションを<aside>タグで囲みます。
広告、注釈、関連記事リンクなどが該当します。
```HTML
<aside>
  <h2>関連記事</h2>
  <ul>
    <li><a href="#">記事1</a></li>
    <li><a href="#">記事2</a></li>
    <li><a href="#">記事3</a></li>
  </ul>
</aside>
```
## footer
主にWebページの最下部に配置するタグです。
著作権情報や連絡先、サイトマップなどを表示するために使用します。
```HTML
<footer>
  <p>Copyright ©2024 Code Lesson All Rights Reserved.</p>
</footer>
```
## div
「div」は「division」の略で、要素をグループ化するために使用
```HTML
<div>
</div>
```
# 表：table
```HTML
<table border="1" style="border-collapse: collapse">
  <tr>
    <td>セル1</td>
    <td>セル2</td>
  </tr>
  <tr>
    <td>セル3</td>
    <td>セル4</td>
  </tr>
</table>
```
- <tr>は、表（テーブル）の中で行を表すタグ
- <th>タグは、表の見出しとなるセルを作成
  - <th>タグを使った表の見出しは、行と列両方に使用できます。
  - <th>タグ内にはテキストの他、<img>などのさまざまな要素を入れる事が可能です。
```HTML
<table>
  <caption>果物の特性</caption>
  <thead>
    <tr>
      <th>名前</th>
      <th>色</th>
      <th>味</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>レモン</td>
      <td>黄色</td>
      <td>すっぱい</td>
    </tr>
    <tr>
      <td>桃</td>
      <td>ピンク</td>
      <td>甘い</td>
    </tr>
  </tbody>
</table>
```
- <thead>（table header）は、HTMLテーブルにおいて表のヘッダー部分をグループ化するためのタグです。このタグを使用することで、テーブルの構造をより明確にし、視覚的にも論理的にもデータの区分をはっきりさせることができます。
  - 構造の明確化: <thead>を使用することで、テーブルのヘッダー部分が明確に区別され、データの理解が容易になります。
  - アクセシビリティの向上: スクリーンリーダーなどの支援技術は、<thead>を認識して、ユーザーにテーブルの構造をよりわかりやすく伝えることができます。
## 表の中で、指定した複数のセルを縦方向に結合する：rowspan
HTMLのテーブル要素で使用される属性の一つ。主に<th>や<td>の属性として使用します。
- rowspan属性を使う際は、colspan属性との組み合わせに注意が必要です。rowspanでセルを結合すると、他の行のセルの配置が変わるため、全体のテーブル構造が崩れることがあります。また、結合したセルの下にあるセルは、結合した分だけ減らす必要があります。正しい行数を考慮して設計しましょう。
```HTML
<td rowspan="結合する行数">結合するセルの内容</td>
```
## 表の中で、指定した複数のセルを横方向に結合する：colspan
colspan属性の値は、結合したい列の数を指定します。例えば、3列目と4列目を結合する場合、colspan="2"とします。これは、2つの列を1つのセルにまとめることを意味します。テーブルの構造を見て、結合したい列の数を数えれば大丈夫です。
# 用語やアイテムの説明をまとめて表示するために使用するHTMLのタグ:dl(Definition List)
```HTML
<dl>
  <dt>用語1</dt>
  <dd>用語1の説明</dd>
  <dt>用語2</dt>
  <dd>用語2の説明</dd>
  <dt>用語3</dt>
  <dd>用語3の説明</dd>
</dl>
```
-  <dt>タグ内には、複数の<dd>タグを指定してはならない

# 入力欄
複数のセレクタをコンマ（,）で区切ることで、それらに同じCSSを適用することができます。
これを用いて、<input>要素と<textarea>要素には同じCSSを適用していきましょう。
## １行のテキスト入力を受け取るための要素：<input>
- 終了タグが不要
- type属性を指定することができ、type属性にsubmitを指定すると、入力欄ではなく、下の図のような送信ボタンになります。
ボタンに表示されるテキストは、日本語のブラウザではデフォルトで「送信」になります。
- ボタンに表示されるテキストを変更することも可能です。
value属性に任意の値を指定することで、ボタンに表示されるテキストを変更することができます
```HTML
<input class = "contact-submit" type = "submit" value = "送信">
```
