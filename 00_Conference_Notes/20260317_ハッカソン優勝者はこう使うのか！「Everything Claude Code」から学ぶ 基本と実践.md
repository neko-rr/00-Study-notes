- Q&A　物足りないポイントありますか？

  - 作者も全部を必要として書いているとは思えず、パッケージとして用意していると思う
  - 部分分けして、必要分だけインストールできるように用意していて欲しいかな
  - 自分と比較してどうですか？

      - 話者は軽量、むしろ固め過ぎると駄目。紹介しているのは、ハーネスをしっかりさせて、行う。
