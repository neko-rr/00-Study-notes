# Claude Code と回す、YouTube運営のPDCA 〜コマンド1発で始まるエージェントオーケストレーション〜	masaya_nishigaki
- YouTube運営
  - Cloud codeのリリースと共に登録者数、急成長
- 企画⇒製作・公開⇒分析⇒改善
  - 15個以上の定型作業を動画ごとに繰り返す
- 例：動画公開・ショート化に必要なコマンドは1発
  - スキルとサブエージェント11体に移譲し、3か所を並列化してShorts作成まで一気通貫
  - 特にタイトル戦略が重要
    - そこにトークン費やすし、良いモデルを使用する
    - 各所から分析して決定する
  - メタデータの生成に対して、別エージェントでメタ検証させる
  - コミュニティの告知も
- 結構上記構成のskillが多い
  - 順序制御して、サブエージェントが各種skill使用して実行
  - 何かを実行開始だけをskillにしている
  - 手作業は、TODOチェックリストに入れている
  - レビューは、HTMLで完結
    - 採用ボタンやレビューの次へ進むが固定
- 分析は、見るべき数字だけ
  - 自作ダッシュボード
  - 判断を鈍らせる間接指標は、表示しない
  - 「○○まで5000人突破」とかが目標
- プラットフォームや目的が変わっても転用できる仕組み
