# 鼎談：アジャイルコーチって何者？
- アジャイルコーチの役割とは？
  - アジャイル以外の事も相談して良い
  - チームの人が自信を持てていないと、普通のコーチングのようになる
    - リーフレーミング
- アジャイルコーチの知名度増えていますか？
  - あんまり聞かない
  - 10年以上やっていると、たまに分かる事がある
- アジャイルコーチの効果をどう説明しますか？
  - チームの変化を見ましょうと言う
    - 自分達で見に行った方が良いですよーと言う
    - アドバイスはする。ここら辺何か問題ありそうですよー
  - 変化が一番だと思う
    - 2か月後のフォローアップ会を開く
      - 上司も立ち会って、こんなに変わりましたよーと見せる
  - 失敗談
    - 自分のした事を並べて、「変化」ではない報告ばかりした
  - 必ずしも変わる事が良い訳ではない
- 大切にしていること、こだわり
  - 支援している時は、「コーチ」である事を忘れて、チームの一員として入る
    - 「コーチ」だからと変な答えを出そうとしてしまった
    - 今、ここにあるベストを探す意識をする
  - 顧客やチームにとって良い人になる（教科書的答え）
  - 今と、違う答えを持てるようになるように促す
    - やった結果、個人が別の道に進んでも良いと思ってやっている
  - 非暴力
    - 人にレッテルを貼って、変えさせようとしない
      - 例：振り返りで、「お客さんの信頼を失った」という前提で話す
  - 事実と推論を分けて話そう
    - 他人の感情を決めつけない
  - 熊狩りで人の臭いを消すように「コーチ」が居るという気配を消す時もある
  - 面倒くさい人だと思われない雰囲気を作る
- リモート？
  - 殆どリモート（2人）
  - 半々（1人）
  - 東京は、リモートが多い
    - 東京で採用出来ずに、あちらこちらの地方でチーム組んでいたりする
- アジャイルコーチとして、上手くいかなかったこと、苦い思い出は？
  - 良くない事をズバズバ言い過ぎた
    - 良くない事を気付かせるのが役目で、勇気をもって言うのが「役目」だと思っていた
      - 上記だけだと、ただの声の大きい告発者みたいになって、誰も動かない
  - スクラムマスターの方が初めての60歳の方
    - 立ち上げに苦労した
    - 学習用の3ヶ月開発
    - スクラムマスターに時間をかけすぎた
      - 開発者から「これではスクラム」の勉強にならないから、スクラムマスターを変える提案が出されていた
  - 「自分だったらこうするのに、なぜこうしない」が表に出過ぎた（若い時）
    - 自分だったらがあるから、感情的に腹立つ
      - 言い方を気を付けるようになった
        - 自分の思ったことの表に出す割合を調節したり
      - 現在、腹立たないようになった
  - 言うか言わないか迷った時は、今これを飲み込んだら良い事があるかを考える
    - 更に言い方を考える
  - 自分がコーチから言われたのが、「お前がそれを言う事で、ドヤりたいだけなら言うな」
  - 一撃必殺
    - 我慢して我慢して言う
  - その場で理解してもらう必要は無い
- アジャイルコーチ同士、どう学び合っている？
  - 社内にアジャイルコーチが複数人居るので、時々お互いに話す機会が定期的にある
  - 基本は、バディで入るから、帰りに2人で話し合う
    - 始まる前に作戦会議して、帰宅時に感想戦
      - 自分は厳しめやフォロー等の作戦
  - 週1、火曜日の朝7:30から集まる（チームが始まる前）
    - ブラックと言われるが、その分他で休んでいる
    - 自分の関心ド真ん中でない事を全身で聞いて、聞いた事を新鮮な内に実践する
  - お客さんのマネジメントの視線を学ぶ
  - コミュニティ
- 生成AIは、アジャイルコーチに関わってくる？
  - 難しい本を読む時に使用
  - スライドの作り方が変わった
    - 話す事を下書きして、○分で収まるか聞いて、スライドを生成
      - 話す内容は、全部自作
  - 壁打ち
    - 上手く言語化できない時に、永遠ラリーして、しっくり来るまで捏ねる
    - 言語化して、再現性を得るために
  - 音声入力で客に話す事を相談
- 地方在住なので、どうやってコミュニティを見つけましたか？
  - スクフェス
  - 自分で作る
- 明日から役に立つクロージング
  - 今のやり方を疑う
    - いつから、やり方を変えてないんだっけと振り返る
    - もう必要ない事をやっているかも
  - 「きっとこう」と決めつけて入ると失敗する
    - 問いかけから始める
  - 挨拶
