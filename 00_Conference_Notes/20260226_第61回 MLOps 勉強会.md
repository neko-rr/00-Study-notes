- RAG評価の基本
  - RAG Triad（TruLens）
    - Context Relevance（検索の質）
      - クエリに関連しているか？
    - Groundedness（根拠）
      - ハルシネーション防止
    - Answer Relevance（回答の質）
      - 質問に答えているか？
    - 海外で良く使用される
  - GPAフレームワーク
    - RAG Triadの発展形
      - Goal
        - Goal Fulfillment（GF）最終的な目的達成度
      - Plan
        - Plan Quality（PQ）計画の最適性。必要なステップの網羅
        - Plan Adherence（PA）計画通りに行動したか
      - Act
        - Logical Consistency（LC）行動の一貫性と論理性。
        - Execution Efficiency（EE）無駄なステップがないか
        - Tool Selection:適切なツールを選べたか
      - OSSとしても提供中
