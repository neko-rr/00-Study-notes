# J-Quants　アップデート
- 【マケデコについて】
- Market API Developer Community (略称「マケデコ」、ハッシュタグ「#マケデコ」) は、Market APIを活用したマーケットの分析や投資を初心者から上級者までDiscord上のコミュニティ活動を通して、学習・実践できる環境を提供することを目的に活動する有志のコミュニティ団体です。
- J-Quantsとは
  - プロ品質のデータを、個人投資家向け
  - 主要な金融データが集約
  - 無料プランから始められる
  - アドオンもある
  - 日本取引所グループ
- 2023年にV1
  - cursorで開示直後の差分を取得（プレミアムプラン）
- J-Quants CLI
  - AIの「手」になる道具
  - 3コマンドで始められる
  - 出力は、CSV・JSON：取得したデータをそのままAIに渡せる
- TDnet 適時開示アドオン
  - 決算短信・開示資料をAPIで取得
- 財務情報の随時更新
  - 開示直後のデータをASが取れる
    - 差分取得
    - 開示直後のデータをAIが取れる＝鮮度
    - 速報分析のワークフローの土台
