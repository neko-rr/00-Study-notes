# 株式会社CARTA HOLDINGS　鈴木 進也	AIに自律的に開発をさせる
- データ基盤を整備しているような会社
- 課題を解決するHowの検討から実装まで全部やらせる
  - Claude CodeのSkills　＋　Sub Agent
- AIに多くを任せると起こる悩み
  - AIのアウトプットが人間の期待と違う
    - ハーネスエンジニアリング
    - プロンプト改善
- 課題の書き方が重要な理由
  - 与えられた課題に対して適切なHowを考えるには、調べてもわからない課題特有の情報が必要
  - 具体的なHowは調べればいい
    - どのHowが適しているかは、AIにやらせるために課題の言語化が必要
- 人間がやってた頃と変わらなくね？
  - 手を動かす主体が変わっただけ
  - 「あとは任せる」レベルまで持っていく
