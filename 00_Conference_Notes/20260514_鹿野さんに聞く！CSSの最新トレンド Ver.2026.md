# 内容
- 資料は、講演後アップロード
- ブラウザサポートの情報
  - Newly available：すべてで対応した機能
  - Widely available：上記になってから、30か月後
    - 最新過ぎて、お客さん対応できるか気にするならば、これを目安にする所が多い
  - もう1個：まだ対応していない所があるよー
- あちこちでベースラインを確認できる
  - 話者が好きなのは、web.dev
- 全対応を中心に今回紹介
- グラデーションを綺麗に見せる（テキストグラデーション）
  - gradient-text--hero：ボックスにグラデーション
  - background-clip:text;
  - color:taransparent;
  - in oklch　を設定すると、グラデーションが目に見えて、鮮やかになる
    - SRGB：中央が暗い反転
    - OKLAB：中央、ほぼ灰色
    - OKLCH：中央、鮮やかな何かの色
- ヘッダーにテーマ切り替え
  - 従来は、Popover難しかった
    - ジャバスクリプト多用やし、位置指定設定も複雑
  - 今、HTMLとCSSだけで可能
  - commandforとcommand
  - @starting-style　でアニメーション可能：表示時
    - allow-discreteの効果でアニメーション：非表示
      - 非表示だけFirefox以外対応済み
  - 細かく説明して、実演   
- カードデザインの要素の高さを揃える
   - 親のカードも子供のカードもdisplay:grid
   - 子のカードをgrid-template-rows:subgridにするのがポイントで、grid-row:span4;とすると、親の4行分という意味になり、カード2枚でも揃う
   - それぞれのカードの高さの最大で揃う
     - 題名が1行か2行かで自動で行幅確保
- Q&A CSSが期待通りじゃなかった時のデバック方法
  - devツールで直接変更して試す
  - AIエージェントにスクショ渡して、依頼
    - Cloudeならば、devツールMCPを使用して調査してくれる
- Q&A AIエージェントにどう指示すれば、良いUIなるか
  - デザイン.mdに書く内容を決めて、書く
