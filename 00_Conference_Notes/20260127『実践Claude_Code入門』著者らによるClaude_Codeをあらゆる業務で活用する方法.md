# Claude Codeの業務活用の概要
## 大嶋 勇樹
- 株式会社ジェネラティブエージェンツ
  - AIエージェントが「ハブ」となり、人間とAIエージェント～
  - 創業2年くらい
- 今日の話
  - ソフトウェア開発以外に使用可能
  - ソフトウェア開発は無い！
  - AIエージェントを作ろうとしている方に是非
- 実装の手順をまとめて、実装させるように「カスタマイズ可能な汎用エージェント」として活用可能
  - コマンド実行やスクリプト読み込んだり、API叩かせやすいからカスタマイズ性が良い
  - フォルダに業務のマニュアルを順番に置いとく
    - マニュアルのバージョン管理にGit
  - 例：議事録作成
- 普通ではできない事
  - 特定のイベントが発生したら（例：メールがきたら）エージェントを実行
  - 特定の時間になったら、自動で動く
  - 一連のタスクをそれぞれ人間と対話しながら業務を遂行する
  - 業務に必要なデータをスプレッドシートのような表形式で読み書き
- 上記出来ない事をするための独自プラットフォームを作って、回しています
## 西見 公宏

  - 自動化は「構造の細分化」、人は「本質の形式化」で動く
  - 自動化は「独立した個」、人は「関係性の中」で機能する
- ずっとメンテナンスし続ける必要がある
  - 「状況Aに対する手順B」を作成し続ける必要がある
  - 感想：この人が完璧求め過ぎるだけでは？
- 本当に人みたいに動かしている
  - 日報とかがノリ良く面白い
  - 情報収集させている：アップデートやニュースや
