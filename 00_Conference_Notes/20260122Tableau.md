# データ可視化とは	Salesforce社　輪﨑 博司氏
- BIツールが生まれた背景
  - データがここ5年で3倍となっているので、Excelでは管理できなくなってきている
  - 脳の仕組み的に「表」を見せられると前頭葉が動いて「考える」が、グラフを見ると後頭葉が動いて「見る」
  - ビジュアライズすると、より早く＆より分かりやすい
- Tableau
  - 2019年よりセールスフォースに買収されて、ファミリーの一員となる
  - **誰でも**分かりやすくする
  - アートに拘ったグラフから、早く簡単なグラフまで多様にできる
  - Tableau Pulse：自然言語で昨日の変化点を教えてくれる
  - Tableau Agent：対話
 - 一見して4社、どれも変わらない
   - 差別化は、セルフBI
     - 使っていて楽しい
     - 学習しやすい
     - 仲間がいる
   - 作成してしまえば、どれも一緒だが、セルフで作成しやすい
   - 従来だと、Excel⇒外部委託でBI作成⇒変更したくなり、見なくなる⇒とにかくダウンロードさせてくれ⇒Excel回帰

 # Tableauを実際に触ってみよう	酒井佑太さん
- Tableau：9年歴くらい
- コンペやハッカソンによく出ている
  - Vix Games　2025/11/20優勝
- ハンズオン：IPAのデータ
  - なかなかきついデータ
    - ヘッダーが2行に分かれている
    - シートが都道府県
    - 数値が混同
  - 今回は、クロウドでバイブコーディングで分離させました
  - 色やサイズを決めると出てくるのが、Excelと異なる所
    - 凡例から右クリックで「除外」で簡単に整えられる：フィルター
    - 「除外」の方が修正しやすく、取り回しが良い
    - 非表示との違いは？
      - あまり、オススメしない。複雑な計算する時だけ良いケースがあるが、ほとんどは、良くない。
  - シート名で何をしたかを書いとくのが、よくある管理法
  - 無料版でも資料に使用可能
    - スクショしようが何しようがOK（セールスフォースの方のお墨付き）
  - 地図で表現が、Excelではできない。Tableauらしい機能
  - ダッシュボードは、今まで作成してきた「シート」を組み合わせれる画面
# クイズなどインタラクティブな講義	酒井佑太さん

- Tableauは、ベン図を選択して、データを統合とかも簡単にできる
